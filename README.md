## 🧭 这是什么

**Vibe Coding**（氛围编程）＝ 主要通过向 AI 编程工具描述需求、把想法直接做成能跑的产品。这份清单专收**中文独立开发者用 Vibe Coding 做出来的真实作品**，社区投稿、按时间持续更新。

**入选标准**

- ✅ 能打开、能用的**产品**：网站 / App / 小程序 / 桌面工具 / 浏览器插件
- ✅ **独立 / 个人开发者**作品（一个人或很小的团队）
- ✅ 开发过程**大量借助 AI 编程**（Cursor / Claude Code / Codex / Copilot …）
- ✅ **面向中文圈**
- ❌ 不收：教程、课程、资源合集、纯代码库、纯开发者工具

**收录什么？**　中文独立开发者用 vibe coding 做的、能打开能用的产品。不收教程 / 资源合集 / 纯代码库 / 纯开发者工具。

**怎么投稿？**　[提 Issue 填表](../../issues/new?template=submit-project.yml) 或发 PR，见 [CONTRIBUTING](./CONTRIBUTING.md)。

**用哪些工具算？**　Cursor、Claude Code、Codex、Copilot、Trae、Windsurf、v0、Bolt、Lovable 等都算。
