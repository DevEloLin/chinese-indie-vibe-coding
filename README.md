## 🧭 这是什么 & 入选标准

**Vibe Coding**（氛围编程）指主要通过向 AI 编程工具描述需求，把想法直接做成能运行的产品，而不必手写全部代码。这份清单专收**中文独立开发者用 Vibe Coding 做出来的真实作品**，社区投稿、持续更新。

**入选标准**

1. **是能用的产品**：网站 / App / 小程序 / 桌面工具 / 浏览器插件，有一个**能打开的链接**。不收教程、课程、资源合集、纯 library、纯开发者工具。
2. **独立 / 个人开发者作品**：一个人或很小的团队，不是公司产品线。
3. **Vibe Coding 做的**：开发过程大量借助 AI 编程（欢迎注明用了什么工具）。
4. **面向中文圈**：中文开发者，或主要服务中文用户的产品优先。

**图例**：`:white_check_mark:` 在运营 ｜ `:clock8:` 开发中·公测 ｜ `:x:` 已关闭

**条目格式**

```
* :white_check_mark: [产品名](https://链接)：一句让人想点进去的介绍。🛠 Cursor/Claude Code · [仓库](https://github.com/...)
```

## ❓ 常见问题（FAQ）

**Vibe Coding 是什么？**
Vibe Coding（氛围编程）指主要通过向 AI 编程工具（Cursor、Claude Code、Codex、GitHub Copilot、Trae、Windsurf 等）描述需求，把想法直接做成能运行的软件。2025 年由 Andrej Karpathy 提出后迅速流行。

**这个清单收录什么？**
只收**中文独立开发者用 vibe coding 做出来、能打开能用的产品**——网站 / App / 小程序 / 桌面工具 / 浏览器插件。不收教程、课程、资源合集、纯代码库或纯开发者工具。

**怎么投稿 / 怎么提 PR？**
提 [Issue 填投稿表单](../../issues/new?template=submit-project.yml)，或直接发 Pull Request。详见 [CONTRIBUTING.md](./CONTRIBUTING.md)。

**用哪些 AI 工具做的算 vibe coding？**
Cursor、Claude Code、Codex、GitHub Copilot、Trae、Windsurf、v0、Bolt、Lovable 等 AI 辅助编程工具都算。

**Chinese Indie Vibe Coding** is a community-curated showcase of real, usable products **vibe-coded by Chinese independent (indie) developers** — built with AI coding tools such as Cursor, Claude Code, Codex, and GitHub Copilot. ## ⚙️ 维护与自动化（不使用 GitHub Actions）

收稿处理由 AI 技能 [`.claude/skills/process-submissions`](./.claude/skills/process-submissions/SKILL.md) 自主完成：校验（链接存活 + 品类符合 + 归一化去重）→ 只增不删加进清单 → 感谢并关闭 Issue / 合并 PR。

本仓库刻意不跑 GitHub Actions，技能通过**托管 Claude Routine 定时跑**（零 Actions）或**维护者本地** `claude "处理投稿"` 运行。链接/格式/去重也可本地自查：`python scripts/validate_list.py`。
